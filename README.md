# CodeAlpha – To-Do List Application

A modern To-Do List application built as part of the CodeAlpha Internship.

### Features
- Add new tasks
- Mark tasks as completed
- Edit and delete tasks
- Persistent storage using browser localStorage

### Tech Stack
- React
- JavaScript
- HTML
- CSS

### Concepts Used
- CRUD operations
- Event handling (onClick, onChange)
- Local storage
